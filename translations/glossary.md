# Bảng thuật ngữ

| Thuật ngữ | Cách dịch/ghi chú |
|---|---|
| Cosmetic products | Sản phẩm mỹ phẩm |
| Formulation | Formulation; có thể dịch là thiết kế công thức/công thức khi ngữ cảnh cần tự nhiên hơn |
| Insight | Insight; hiểu biết sâu về người tiêu dùng khi cần giải thích |
| Claim | Claim; tuyên bố/khẳng định sản phẩm khi cần diễn giải |
| Verbatim | Verbatim; phát ngôn/trích dẫn nguyên văn từ người tiêu dùng khi cần diễn giải |
| Colour psychology | Tâm lý học màu sắc |
| Chromatic accord | Accord màu sắc; tổ hợp/phối hợp màu sắc khi cần diễn giải |
| Sensory | Cảm quan; thuộc về giác quan |
| Eye-tracking | Eye-tracking; theo dõi mắt khi cần diễn giải |
| EEG | EEG; điện não đồ khi cần diễn giải |
| fMRI | fMRI; chụp cộng hưởng từ chức năng |
| Saccade | Saccade; chuyển động mắt nhanh giữa các điểm cố định thị giác |
| Electrodermal activity | Hoạt động điện da |
| Heat map | Heat map; bản đồ nhiệt khi cần diễn giải |
| Soft-focus | Soft-focus; hiệu ứng làm mờ quang học/nếp nhăn khi cần diễn giải |
| Explicit evaluation | Đánh giá hiển hiện |
| Implicit evaluation | Đánh giá ngầm định |
| Concept | Concept; ý tưởng/khái niệm sản phẩm khi cần diễn giải |
| Packaging | Bao bì |
| Sensory additivity | Tính cộng hưởng cảm quan |
| Sensory suppression | Sự ức chế cảm quan |
| Multisensory process | Quá trình đa cảm quan |
| Olfactory note | Nốt hương |
| Aromacology | Aromacology; lĩnh vực nghiên cứu ảnh hưởng của mùi hương lên cảm xúc/hành vi |
| Neuromarketing | Neuromarketing; marketing thần kinh khi cần diễn giải |
| Synesthesia | Cảm giác kèm |
